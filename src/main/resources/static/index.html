请求助手

选择模板后会按说明文档生成一份可直接发送的请求。
建议联调顺序：先发送 /api/device/sync，再调 /api/unified/command。 deviceId 表示 QDMS 平台设备 ID，QDNS 会根据 Redis / 节点缓存中的 deviceIp + devicePort 去定位下游设备。 对于 10007 upgrade 和 10008 reboot，QDMS 侧不应只看受理响应，必须继续轮询 /api/unified/task?taskId=...。
