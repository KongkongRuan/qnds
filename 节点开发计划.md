# QDNS 服务节点开发计划

> 版本: V1.0
> 日期: 2026-03-25

## 一、项目概述

QDNS（QVPN Device Node Service）是QVPN集中管理平台的服务节点组件，负责通过SNMP协议与VPN设备通讯，定时扫描设备状态并存入Redis供管理平台读取。

## 二、技术架构

| 项目 | 技术选型 |
|------|---------|
| JDK | Java 1.8 |
| 框架 | Spring Boot 2.7.18 |
| 微服务 | Spring Cloud 2021.x + Nacos 2021.0.5.0 |
| SNMP | SNMP4J 3.8.2 |
| 缓存 | Redis (Spring Data Redis) |
| 序列化 | Jackson |
| 前端 | 原生HTML/CSS/JS (内嵌调试界面) |

## 七、运行说明

### 7.1 依赖
- JDK 1.8+
- Redis (可选，无Redis时可使用本地设备)
- Nacos (可选，可通过环境变量NACOS_ENABLED=true启用)

### 7.2 编译
```bash
cd D:\python\project\qdns
mvn clean package -DskipTests
```

### 7.3 运行
```bash
java -jar target/qdns-1.0.0.jar
```

### 7.4 访问
- 调试界面: http://localhost:18023/
- API文档: http://localhost:18023/api/node/info

### 7.5 配合VPN-Sim使用
1. 启动VPN-Sim模拟器 (192.168.5.x)
2. 启动QDNS服务节点
3. 打开调试界面 http://localhost:18023/
4. 点击 "添加VPN-Sim设备" 快捷按钮
5. 点击 "立即采集" 查看结果
